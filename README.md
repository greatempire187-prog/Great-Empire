## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js (for backend functionality)
- MongoDB (for database operations)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/greatempire187-prog/great-empire-portfolio.git
   cd great-empire-portfolio
   ```

2. **Frontend Setup**
   - Open `index.html` in your web browser
   - No additional setup required for frontend features

3. **Backend Setup** (Optional - for full functionality)
   ```bash
   cd backend
   npm install
   cp .env.example .env
   # Edit .env with your configuration
   npm start
   ```

4. **Database Setup** (Optional)
   - Install MongoDB
   - Update `MONGODB_URI` in `.env` file
   - The application works without database using localStorage

## Configuration

### Environment Variables
Create a `.env` file in the backend directory:
```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/great-empire
JWT_SECRET=your-secret-key
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
FRONTEND_URL=http://localhost:8000
```

### Settings
All user settings are stored in localStorage and persist across sessions:
- Profile information
- Appearance preferences
- Notification settings
- Privacy controls
- Account security
